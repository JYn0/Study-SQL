다음 테이블을 작성하시오

```SQL
CREATE TABLE CAR(
	ID CHAR(5),
	NAME VARCHAR2(20),
	CSIZE NUMBER(10)
);
```

ID -> PK 변경

CSIZE 5개 변경

REGDATE(DATE) 추가

NAME, CSIZE, REGDATE를 NOT NULL로 변경

컬러명 CSIZE를 CSIZES로 변경

테이블 명을 T_CAR로 변경

```sql

ALTER TABLE CAR ADD PRIMARY KEY(ID);
ALTER TABLE CAR MODIFY(CSIZE NUMBER(5));
ALTER TABLE CAR ADD (REGDATE DATE);
ALTER TABLE CAR MODIFY(NAME NOT NULL);
ALTER TABLE CAR MODIFY(CSIZE NOT NULL);
ALTER TABLE CAR MODIFY(REGDATE NOT NULL);
ALTER TABLE CAR RENAME COLUMN CSIZE TO CSIZES;
ALTER TABLE CAR RENAME TO T_CAR;
```
